Grid Layout

CSS Grid allows a user to design web pages using a multi-directional grid based system, removing the need to use more complex floats and positioning

Important Terms

Before beginning to use CSS Grid to change the layout of your webpage, there are a few key terms to understand.

  • Grid Container - The direct parent of all grid items, the Grid Container must have the display:grid property.

  • Grid Item - The direct children (or descendants) of a Grid Container.

  • Grid Row - The cells that make up a horizontal line in a grid.

  • Grid Column - The cells that make up a vertical line in a grid.

  • Grid Gap - The space between the cells in a grid. The gap is determined by the column-gap, row-gap, and gap properties.

  • Grid Area - The cells surrounded by 4 grid lines, any number of cells can be included, but must be surrounded by 4 grid lines (gaps).

Grid Elements

A grid layout consists of a parent element and one or more child elements.

Example


     <div class="container">
       <div class="item">A</div>
       <div class="item">B</div>
       <div class="item">C</div>
       <div class="item">D</div>
       <div class="item">E</div>
       <div class="item">F</div>
     </div>

Parent Properties (Container)

grid-template-columns

grid-template-rows

Defines the columns and rows of the grid. Values are separated by a space and can be entered as percentage, length or fraction measurements. 


     .container{
      grid-template-columns: 5px 10% 1fr;
      grid-template-rows: 10px 5% 1fr;
     }

If your grid has repeating parts, use can use the repeat() value to write less code. 


     .container{
      grid-template-columns: 1fr 1fr 1fr 1fr 1fr;
      grid-template-rows: 1fr 1fr 1fr 1fr 1fr;
     }

     =

     .container{
      grid-template-columns: repeat(5, 1fr);
      grid-template-rows: repeat(5, 1fr);
     }

grid-template-areas

Defines a grid template by referencing names of a grid area which are specified using the grid-area property. Using the same name for multiple areas causes the content to span multiple areas and a period signifies an empty cell. 


      .container{
        grid-template-areas:
        "area-1  area-1  area-2"
        "area-3     .    area-4"
        "area-5  area-6  area-6"
      }

      .item a{
        grid-area: area-1;
      }

      .item b{
        grid-area: area-2;
      }

      .item c{
        grid-area: area-3;
      }

      .item d{
        grid-area: area-4;
      }

      .item e{
        grid-area: area-5;
      }

      .item f{
        grid-area: area-6;
      }

column-gap

row-gap

gap

The space between the cells in a grid. The gap is determined by the column-gap, row-gap, and gap properties. Any value type can be used to specify how large the gap is between cells. 


      .container{
        grid-template-columns: 100px 1rem 1fr;
        grid-template-rows: 50px 1rem 1rem;
        column-gap: 10px;
        row-gap: 1rem;
      }

justify-items

align-items

place-items

Justify-items aligns items vertically within their cell, align-items aligns items horizontally within their cell. Both properties apply to all grid items and accept the following values:

  • Start - aligns items with the start (left, top) edge of their cell.

  • End - aligns items with the end (right, bottom) edge of their cell.

  • Center - aligns items with the center of the cell.

  • Stretch - fills the whole width or height of the cell, this is the default value.

Place-items is a short hand to combine both justify-items and align-items, it accepts both values in one declaration with align-items coming before justify-items. 


        .container{
          justify-items: start | end | center | stretch;
          align-items: start | end | center | stretch;
        }

        .container{
          place-items: center start;
        }
Child Properties (Items)

justify-self

align-self

place-self

Justify-self aligns items vertically within their cell, align-self aligns items horizontally within their cell. Both properties apply to content in a single grid item and accept the following values:

  • Start - aligns grid item with the start (left, top) edge of their cell.

  • End - aligns grid item with the end (right, bottom) edge of their cell.

  • Center - aligns grid item with the center of the cell.

  • Stretch - fills the whole width or height of the cell, this is the default value.

Place-self is a short hand to combine both justify-self and align-self, it accepts both values in one declaration with align-self coming before justify-self. 


        .container{
          justify-self: start | end | center | stretch;
          align-self: start | end | center | stretch;
        }

        .container{
          place-self: center start;
        }
